# Étape 0 - Installation de Docker

L'installation de Docker est gratuite pour un usage personel ou dans le cadre d'une petite structure : [https://www.docker.com/products/personal/](https://www.docker.com/products/docker-desktop/). 

Vous pouvez l'installer sur Linux, Mac ou Windows.

La façon la plus simple est d'utiliser [Docker Dekstop](https://www.docker.com/products/docker-desktop/).
